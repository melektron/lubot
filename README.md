# LiveUnderflowBot

A bot to monitor LiveOverflow's minecraft server using discord.

## Current Features
 - none

## Setup

Create src/token.json with contents
```json
{
    "token": "...",
    "botid": "...",
    "connectionData": {
        "host": "...",
        "port": 25565,
        "username": "Player",
        "password": "password"
    }
}
```
